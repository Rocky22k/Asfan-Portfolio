Fruit Market War

A responsive website inspired by Apple's design, exploring UI/UX principles and AI-assisted content generation

Project Overview

Fruit Market War was my first major exploration into advanced CSS techniques and creative web design. Inspired by a MrWhoseTheBoss video spoofing Apple with a fictional "Passionfruit" smartphone brand, I decided to create a website that captured that same satirical energy while pushing my CSS skills to new limits.

This project became a playground for experimenting with gradient effects, lighting techniques, and visual styling that I'd never attempted before. Working with minimal guidance, I used AI assistance to help refine my ideas and implement effects that seemed impossible at first. The experience taught me that CSS could create stunning visual experiences far beyond basic styling.

What I Built

  • Apple-inspired layout with clean, minimalist design principles
  • Custom gradient effects and lighting that became the foundation for my current portfolio
  • Responsive design that adapts seamlessly across different screen sizes
  • Creative integration of images and resources from the original Mr. Whose the Boss content
  • Interactive hover effects and smooth CSS animations

Key Learning Moments

This was my first time diving deep into CSS beyond the basics. I discovered how gradients could transform a simple design into something that feels premium and modern. The lighting effects I experimented with here directly influenced the design system I use throughout my current portfolio.

Working on this project showed me how much creativity you can bring to web development. It wasn't just about making functional pages - it was about creating an experience that feels engaging and visually impressive. This project gave me the confidence to experiment with advanced techniques in future work.
